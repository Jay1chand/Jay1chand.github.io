DTFT Calculator

This tool computes the Discrete-Time Fourier Transform (DTFT) of a given signal \(x(n)\). The DTFT is used to analyze the frequency-domain behavior of discrete-time signals. You can input expressions like delta(n-m), cos(2*pi*f0*n*dt), or sinc(n), and the calculator will return both the original signal representation and its DTFT in symbolic and LaTeX form.


Example: delta(n-m), cos(2*pi*f0*n*dt), sinc(n)
